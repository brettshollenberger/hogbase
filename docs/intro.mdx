## Quick Start

1. Install Hogbase:

```bash
npm install hogbase
```

2. Set up your PostHog key:

```bash
export POSTHOG_API_KEY=your-project-api-key
```

3. Wrap your app with the ExperimentsProvider:

```tsx
import { ExperimentsProvider } from '@hogbase';

const experiments = [
  {
    name: "copy",
    label: "Copy Variations",
    possibleValues: ["time_savings", "customer_focused"]
  }
];

function App() {
  return (
    <ExperimentsProvider experiments={experiments}>
      <YourApp />
    </ExperimentsProvider>
  );
}
```
